Mi Portafolio

TryGoal

Proyecto de bases de datos, con el objetivo de recolectar información de la liga inglesa de fútbol y mostrar los datos por medio de una página web.

Recursos: Figma, React, Heroku y MySQL.

App: TryGoal

Proceso

Componentes

Primero se escogió una paleta de colores, creación de componentes base y un logo.

Diseño de interfaces

Posteriormente se realizaron los diseños de las interfaces gráficas haciendo uso de Figma.

Final

Después de tener todas las interfaces diseñadas, se creó la base de datos, para implementarla dentro del desarrollo Front-end haciendo uso de React.
